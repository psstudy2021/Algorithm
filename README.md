# ⭐️ 1. 협업 도구 및 코드 공유 형식

### 과제 공유 방식

각 주차 directory에 해결한 문제를 push

### 파일 이름 규칙

leetcode_문제이름_이니셜, programmers_문제이름_이니셜, baekjoon_문제이름_이니셜 등
예) programmers_행렬이동하기_jdh.java

### 풀이 시간 기재

예: 1시간 이상 - 초과, 00분~60분 사이

### 스터디 2시간 진행 방식

- 1시간 코드 리뷰 및 QnA
- 30분 공통 문제 풀이
- 30분 공통 문제 리뷰
